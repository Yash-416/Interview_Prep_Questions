Project 

SafeMind is a secure web-based mental health platform that connects users with verified therapists through online appointment booking and real-time WebRTC video consultations.
It solves the problem of limited accessibility, privacy concerns, and delayed support in traditional mental health services, especially for remote users.
The platform allows users to register, book sessions, track moods, complete self-assessments, and maintain encrypted personal diaries, along with an AI chatbot for guidance. Therapists can manage schedules and conduct consultations, while admins verify professionals and monitor operations without accessing sensitive data.
I built the frontend using ReactJS and the backend using Spring Boot / ASP.NET with MySQL as the database. Security is handled using JWT authentication and role-based access control.
The system follows a layered architecture: Controller → Service → Repository → Database. One key challenge was integrating WebRTC securely and implementing strict role-based authorization.
Overall, the platform provides a scalable, privacy-focused digital mental health solution, and in the future I plan to add payment integration and advanced AI-based insights
